# Emotion Detection System

An AI-based emotion detection system that identifies human emotions using facial expressions through machine learning and computer vision techniques. The project uses image processing and deep learning models to classify emotions in real time.

## Features

- Real-time emotion detection
- Facial expression recognition
- Camera or image input support
- Emotion classification using AI/ML
- User-friendly interface
- Model training and evaluation
- Performance metrics visualization

## Detected Emotions

- Happy
- Sad
- Angry
- Fear
- Surprise
- Neutral
- Disgust

## Technologies Used

- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## System Overview

The system captures facial images through a camera or uploaded image, processes the detected face, and predicts the corresponding emotion using a trained machine learning model.

## Dataset

The model was trained using a facial emotion recognition dataset containing labeled facial expressions for different emotions.
